## What is ParquetFrame?

ParquetFrame is a powerful Python library for working with Parquet files, featuring:

- Multi-engine support (pandas, Polars, Dask)
- Entity Framework with decorators
- Zanzibar-style permissions
- YAML-based workflow engine
- Rust acceleration for performance-critical operations
